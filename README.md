# polyglotMemeGenerator

An attemt to connect JS and Ruby in a mad meme mix.

How to run the server:
```sh
$ cd polyglotMemeGenerator
$ node server.js
```

Deployment:

```sh
$ cd polyglotMemeGenerator
$ npm install express
$ npm install body-parser
$ npm install multer
```
